QCaptcha ist eine Anti-Spam Lösung für Webmaster, die sehgeschädigte Menschen nicht ausgrenzt und auf Datenschutz achtet

== Description ==
Eine großes Hindernis im Internet sind für Blinde Captchas, da sie weder Schrift aus Bildern abschreiben oder bestimmte Bilder auswählen können. Außerdem sind die Meisten Audio-Captchas kaum verständlich. QCaptcha setzt deshalb auf das Beantworten von einfachen Fragen, sodass Blinde nicht ausgesperrt werden.

Im Gegensatz zu anderen populären Captcha-Lösungen wie z.B: Google reCAPTCHA werden durch QCaptcha keine Daten gesammelt und an Drittanbieter übertragen. QCaptcha wird vollständig auf Ihrem eigenen Server gehostet, sodass Sie die Kontrolle über die Daten Ihrer Nutzer behalten.

QCaptcha unterstützt aktuell Deutsch, Englisch und Niederländisch.

Verbesserungsvorschläge sind willkommen!
[Zur Plugin-Webseite](https://timokoessler.de/qcaptcha/ "Mehr Informationen").

== Installation ==
1. Laden Sie die neuste Version des QCaptcha Wordpress-Plugins herunter.
2. Melde dich am Admin-Dashbord deines Wordpress Blogs an und öffne die Plugin installieren Seite
3. Lade die zuvor heruntergeladene Datei hoch
4. Passe die Einstellungen des Plugins nach deinen Wünschen an
